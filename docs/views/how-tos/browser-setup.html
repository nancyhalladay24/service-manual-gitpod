Setting up the kit in the browser

If you can't install the kit or prefer not to use the kit locally on your computer, you can set up the kit in your browser instead.

To set up the prototype kit in your browser you will need to use Gitpod. Gitpod creates automated workspaces and dev environments in your browser from a repository on Github.

You can read more about using Gitpod on their website.

1. Create a new repository on your Github account

The first thing you need to do is create a new repository on your Github account.

2. Import the NHS.UK prototype kit repository

Copy this URL:

https://github.com/nhsuk/nhsuk-prototype-kit.git

3. Install Gitpod to your workspace or organisation

If your workspace or organisation already has Gitpod installed you will need to configure your Github repository to Gitpod instead
