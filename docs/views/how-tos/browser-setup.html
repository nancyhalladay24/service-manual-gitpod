Setting up the kit in the browser

If you can't install the kit or prefer not to use the kit locally on your computer, you can set up the kit in your browser instead.

To set up the prototype kit in your browser you will need to use Gitpod. Gitpod creates automated workspaces and dev environments in your browser from a repository on Github.

You can read more about using Gitpod on their website.

1. Import the NHS.UK prototype kit repository to your Github account

Go to 'Import your project to GitHub' on your Github account.

And copy this URL in to the text field labelled 'your old repositories Clone URL':

https://github.com/nhsuk/nhsuk-prototype-kit.git

Then, name your repository and start the import by clicking the green button labelled 'begin import'.

Once the import is complete you will recieve an email from Github.

2. Install Gitpod to your workspace or organisation

Use this Gitpod installer to add the Gitpod app to your Github workspace or organisation.

You will need to:

  1. Open the Gitpod installer. You may be asked to sign in to your Github account.
  2. Select the workspace or organisation you want to add Gitpod to
  3. Choose to install Gitpod to a select repository or to all your repositories. Then click 'install'. You may be asked to enter your Github password.
  4. Sign in to your Github account on Gitpod. To continue you will need to authorise Gitpod to access your account.
  5. Click 'Install app' to install the Github app for Gitpod. To complete installation you will need to authorise Gitpods additional permissions to access your repositories.

If your workspace or organisation already has Gitpod installed you will need to configure your Github repository to Gitpod instead

3. Open Gitpod in your new repository

Go to your new repository and open Gitpod by prefixing the URL with:

gitpod.io/#

For example:

gitpod.io/#https://github.com/nhsuk/service-manual-gitpod

Alternatively, you can install a Gitpod browser extension if you use Chrome or Firefox

4. Install the kit in Gitpod

To install the kit, enter in the Gitpod terminal:

npm install

While installing it may WARN about some items - this is ok. As long as there are no ERRORs you can continue.

5. Run the kit

Start your local server by entering in the terminal:

npm run watch

After the kit has started, you should see a message telling you that the kit is running:

6. Check it works

In your web browser, visit http://localhost:3000 opens in a new tab

You should see the prototype welcome page.

Installation complete

The kit is now installed. Congratulations!
