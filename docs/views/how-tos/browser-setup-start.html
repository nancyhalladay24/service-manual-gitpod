Set up the kit in your browser

If you can't install the kit or prefer not to use the kit locally on your computer, you can set up the kit in your browser instead.

To set up the prototype kit in your browser you will need to use Gitpod. Gitpod creates automated workspaces and dev environments in your browser from a repository on Github.

This guide will show you how to get setup in 6 steps.

You can read more about using Gitpod on their website.

Before you start

You will need a Github account to use Gitpod. You can sign up to Github on their website

If you have any problems, ask a developer on your team, email us or get in touch on the NHS digital service manual #prototype-kit Slack channel.
