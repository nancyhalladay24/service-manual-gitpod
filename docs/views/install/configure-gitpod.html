Configure a Github repository to Gitpod

If your organisation or workspace already has Gitpod installed you can configure Gitpodâ€™s repository access through the Gitpod installer.

You will need to:

  1. Open the Gitpod installer. You may be asked to sign in to your Github account.
  2. Click 'configure' on the organisation or workspace panel link
  3. In the repository access section, select the repository that you want to allow Gitpod to access
  4. Click 'update access' or 'save'

If you're not an admin on the repository you want to use Gitpod on you will need to request the configuration. An admin can approve this in the project under the applications section.

To start your Gitpod environment from a Github repository follow this guidance on Gitpod.io

I don't have the option to configure a repository

You need Gitpod installed on a Github workspace or organisation before you can configure a repository. Follow our guidance on how to install Gitpod to a Github organisation or workspace
