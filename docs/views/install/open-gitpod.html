Step 3 of 5 Open Gitpod in your new repository

Go to your new repository and open Gitpod by prefixing the URL with:

gitpod.io/#

For example:

gitpod.io/#https://github.com/nhsuk/service-manual-gitpod

Alternatively, you can install a Gitpod browser extension if you use Chrome or Firefox
