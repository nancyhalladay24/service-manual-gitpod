Step 2 of 5 Install Gitpod to your Github workspace or organisation

Follow this Gitpod installer to add the Gitpod app to your Github workspace or organisation.

You will need to:

  1. Open the Gitpod installer. You may be asked to sign in to your Github account.
  2. Select the workspace or organisation you want to add Gitpod to
  3. Choose to install Gitpod to a select repository or to all your repositories. Then click 'install'. You may be asked to enter your Github password.
  4. Sign in to your Github account on Gitpod. To continue you will need to authorise Gitpod to access your account.
  5. Click 'Install app' to install the Github app for Gitpod. To complete installation you will need to authorise Gitpods additional permissions to access your repositories.
My workspace or organisation already has Gitpod installed

If your workspace or organisation already has Gitpod installed you can follow our guidance on how to configure a Github repository to Gitpod
