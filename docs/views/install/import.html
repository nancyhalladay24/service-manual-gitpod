Step 1 of 5 Import the NHS.UK prototype kit repository to your Github account

Go to 'Import your project to GitHub' on your Github account and copy this URL in to the text field labelled 'your old repositories clone URL':

https://github.com/nhsuk/nhsuk-prototype-kit.git

Then, name your repository and start the import by clicking the green button labelled 'begin import'.

Once the import is complete you will recieve an email from Github.
